### C Expressions - Exercise 4.15

Give the values of ```i``` and ```j``` after each of the following expression statements has been executed.
(Assume that ```i``` has a value of 1 initially and ```j``` has the value of 2.)

```
(a) i += j;
(b) i--;
(c) i * j / i;
(d) i % ++j;
```

### Solution

(a) 3, 2
(b) 2, 2
(c) 2, 2
(d) 2, 2
