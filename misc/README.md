### gcc compile example 

```
gcc -O -Wall -pedantic -ansi -std=c99 -o 1.out 1.c
```

```
-Wall (all -W options i.e all warnings)
-O (Used in conjunction with -Wall for maximum effect)
-pedantic (Issue all warnings required by the C standard,  reject non standard features)
-ansi (Turns off features of GCC that aren't standard C)
-std=c99 (Use the c99 compiler)
```
